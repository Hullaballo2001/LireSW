
# Présentation 

Ceci est un test de publication de notes Obsidian sur Github avec le plugin `obsidian-mkdocs template` *Publish your public notes with MkDocs*

# Contenu

Il y a plus de 300 romans dans l'univers Star Wars. Pendant le 1er confinement dû à la COVID-19 j'ai commencé par le plus ancien que j'ai trouvé.
Depuis je continue, à mon rythme, de compléter ce petit challenge perso. C'est vraiment pour mon usage personnel et je n'ai pas la prétention que ce soit utile à qui que ce soit. Le but poursuivi dans la publication de ces notes est de tester le plugin ci-dessus en exposant du contenu perso mais non sensible.

## Sur les romans SW

La principale difficulté est de trouver la chronologie qui me permet de les lire dans l'ordre de l'histoire. Wikipédia propose la liste parfaite, à jour et pour le Canon comme pour l'univers Legend (non officiel). Ce qui est très cool mais pourrait mettre à l'épreuve votre motivation c'est que de nouveaux romans sont sans cesse publiés !
